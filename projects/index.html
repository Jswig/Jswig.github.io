A few highlights from my personal projects. You can also find the full list of repositories here on Github. Several of the Kaggle competitons were done in collaboration with other members of Data Science Slugs.

LANL Earthquake Prediction

code

My first Kaggle competition, in which I use stacked models (random forests, support vector regression and k-nearest neighbors) to predict earthquakes, in the top ~5%. A few optimizations enabled me to do all of the training on my laptop. Technologies used include scikit-learn and pandas.

Understanding Clouds from Satellite Data

code

Kaggle competition in which I classified clouds on satellite images using a U-net implemented in TensorFlow. Used customized optimizers and data augmentation to boost performance, deploying training on a combination of cloud VMs and HPC clusters. Technologies used include TensorFlow witht the Keras frontend, pandas and Tensorboard for model debugging. Google Cloud Compute platform was used to supplement the university's compute cluster.

Who is Satoshi?

code | write-up

Open-ended class project where I use text mining in R to determine whether cryptographer Nick Szabo might have written under the name of Satoshi Nakamoto.

Kannada MNIST

write-up

Variation of the classical MNIST dataset using the Kannada script, with a added difficulty of having to predict on a validation set that is significantly different from the training set. Used a CNN architecture to achieve this improved performance, boosting the accuracy on the validation set from 0.5 too 0.75.
